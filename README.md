# React ES6 Reflux boilerplate (node-webkit, gulp, webpack)

## About
node-webkit<br />
gulp<br />
webpack<br />
react es6<br />
babel es7 annotations<br />
reflux<br />
react hot loader (for webpack)

## Install
```
$ npm install
```

# 3 Ways to Use:
## Start with Gulp
```
$ gulp
$ http://localhost:1338
```

## Start with webpack
```
$ npm start
$ http://localhost:3000
```

## Start with node-webkit 
```
$ gulp
$ nw .
```
